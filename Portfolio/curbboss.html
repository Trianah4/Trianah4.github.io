Project: CurbBoss Dashboard


Project Type: Client


Role: UI/UX Design


  • Research
  • Interaction Design
  • Visual Design

Overview

CurbBoss is a Saas Company that offers a parking management platform to help companies of all sizes and residential areas manage end-to-end parking.

The Goal

The goal was to create a flexible dashboard that allows for straightforward entry of vehicles, handling of registered vehicles, being able to add/remove/edit vehicle data, enforcement/citations, payment, and visitor validation.

Startoff

To start off, different iterations of sketches were done to try and find the best way to fit the clients wants and needs for this dashboard. I then had to ask myself a few questions:

Is this information provided coherent and easy to understand/navigate through the dashboard? and how can the user experience be straightforward and easy for those who are familiar with and aren't too familiar with using a dashboard?

Asking these questions and understanding the clients wants and needs helped me understand the information that should be found on these pages and how to develop the user experience for this dashboard.

Information Architecture

Mid fidelity Wireframes

These mid-fidelity wireframes helped with figuring out how to structure the dashboard to the clients desire while understanding how to make the user experience intuitive.

High fidelity Wireframes

Seamless Navigation & Consistency

Before establishing a familiar side navigation for the dashboard, there needed to be an understanding how the user is able to access different data on the same page as well as having the ability to to create a new entry without having to navigate outside of that page.

It was decided that the pages would use a sub navigation tab menu to access data seamlessly without having the hassle of navigating to another page to access information or to create new entries. If the user needs to add a new entry, a button on that same page will open a modal window which will then allow you to input the information for the new entry.

This allows for the dashboard to maintain its consistency throughout each page and makes it easy for users to navigate through it.

Notifications

One feature the client requested was to have an announcements page/being able to post announcements. Deciding to take it one step further and creating a Notifications drop down menu would allow users to see all the activity that other users have added, changed, edited or announcements that have been posted. These notifications would appear in different states to differentiate between being read and unread to make it easier for users to understand.

Filter

The filter feature will allow users to filter out by the day presets that are listed or they can choose a custom filter. Users are also allowed to search at a specific time of day. This will provide ease when filtering out data they’re looking for.

Style Guide

A simple blue tone was utilized to give it a cool and calm feeling. It went well with the modern minimal design feel that was trying to be achieved. The main typeface that was used was Roboto, it gave it a professional feel and was extremely versatile between the styling.

Residential Dashboard

Aside from designing the parking management dashboard, the client requested a design for a residential parking dashboard for those residential communities that are under a HOA and need to register their own vehicle or register a guest parking pass. There are a few differences between both dashboards but was able to keep in my mind how to implement the same user experience between both dashboards.

Takeaways

As someone who has used a dashboard in the past it was an interesting process trying to design one while keeping user experience in mind. Understanding the parking data that users will need to access, view, edit, input was a challenge. There were other features and designs that I wanted to flesh out by doing surveys, interviews and user testing but was limited due to time constraints, even though those methods were not used they will be incorporated in future projects.
